# Registry's TodoList

### This is simple TodoList Web Application

### Author: RegistryDev

<br />

## Installation

### Assuming npm is installed

```zsh
git clone https://github.com/RegistryDev/todo-list.git

cd todo-list

npm i
```

### And then You have to write .env.local

```zsh
sudo nano .env.local

NEXT_PUBLIC_API_URL = http://localhost:3000
```

### Start Dev Server

```zsh
npm run dev
```
